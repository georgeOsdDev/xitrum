LocalLRUCache

class LocalLRUCache[K, V] extends LinkedHashMap[K, V]

Non-threadsafe, non-distributed LRU cache.

See: http://littletechsecrets.wordpress.com/2008/11/16/simple-lru-cache-in-java/ http://amix.dk/blog/post/19465

Xitrum uses this for storing etags for static files. Each web server in a cluster has its own cache of (path, mtime) -> etag.
